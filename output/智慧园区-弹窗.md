## 配置项

- ```javascript
  {
  	// 列表资产ID
      资产ID：'9d1efdbe-01b3-3054-952b-15a92b23f84c'
      
      // 用于决定列表是单选还是多选
      列表类型：
      
      // 选择数据后，展示在输入框的数据字段
      回填字段：'instrument'
      
      // 用于拼接回填字段的符号
      拼接符号：'-'
      
      // 用于决定列表所需要展示的列，需要是JSON格式
      展示字段：[{"key":"instrument","label":"仪表号"},{"key":"park_name","label":"园区名称"}]
  }
  ```

## 逻辑控制

- ##### 事件(events)

  > 事件名称：`内容改变`

  > 事件传参：

  ```javascript
  [
       {
         name: "内容",
         key: "value",
         dataType: "objectArrary",
       },
  ]
  ```
  
- ##### 动作(actions)

  > 事件名称：`过滤数据`
  
  > 事件传参：
  
  ```js
  [
  	{
        key: "value",
        name: "过滤条件",
        dataType: "object",
      },
  ]
  ```
